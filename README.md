# GLS UNCODED

**GLS UNCODED** is the official hackathon website developed for the university's annual hackathon. It serves as the central platform for participants to explore the event, register their teams, view schedules, learn about tracks, and stay updated throughout the competition.

The name **"UNCODED"** represents ideas that are yet to be built, encouraging innovation, creativity, and problem solving.

---

## Features

* Modern landing page
* Event information and timeline
* Team registration
* Hackathon tracks and themes
* Rules and guidelines
* FAQ section
* Sponsor showcase
* Responsive design
* Smooth animations and interactive UI

## Installation

```bash
git clone https://github.com/your-username/uncoded.git
cd uncoded
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```
